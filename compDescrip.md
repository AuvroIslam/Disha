# Description

The Build with Gemma 4 Hackathon invites university students across Bangladesh to build innovative AI applications using **Gemma 4**, Google's latest family of open models.

Participants may solve problems in any domain, including:

* Education
* Healthcare
* Accessibility
* Productivity
* Agriculture
* Finance
* Climate
* Public Services
* Social Impact
* Open Innovation

Projects must use **Gemma 4 as the primary AI model** and demonstrate meaningful integration into the application.

---

# Goal

Build an AI application powered by **Gemma 4** that solves a real-world problem.

Your submission should clearly demonstrate:

1. The problem you are solving.
2. Why the problem matters.
3. How Gemma 4 is integrated.
4. Technical implementation.
5. System architecture.
6. Working prototype or live demonstration.
7. Real-world impact.

# Additional Rules

## Gemma Requirement

Projects **must** use **Gemma 4** as the primary AI model.

Gemma 4 must also be the **only Large Language Model (LLM)** used.

---

## Allowed Technologies

These are permitted as supporting technologies:

* Traditional Machine Learning
* Computer Vision models
* OCR
* Speech Processing
* Databases
* Vector Databases
* APIs
* Search systems
* Non-generative AI models
* Other supporting tools

These technologies **must not replace Gemma 4** as the application's primary intelligence.

---

## Not Allowed

The following are **not permitted**:

* Any other Large Language Model (LLM)
* Any other Generative Foundation Model

Projects violating this rule may be disqualified.
